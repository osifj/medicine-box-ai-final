# Local Medicine Box Dataset Plan

## Goal

Train a single-class object detection model (`medicine_box`) for Grove Vision AI V2.

## Data Collection

### Option A: Re-annotate existing images (Faster)

The old project has 96 images at `/Users/dep/Projects/New_project/captures/host_session_001/images/` that show medicine boxes. These can be re-annotated with whole-box labels instead of current barcode/text sub-region labels.

### Option B: Capture new images (More variety)

Use the Grove Vision camera or any USB/webcam to capture medicine boxes in various conditions.

### Collection Guidelines

1. **Subject**: One or more medicine boxes (with packaging, labels visible)
2. **Class**: Only `medicine_box` — annotate the entire box outline
3. **Minimum**: 50-100 images for a working model, 200+ for better results
4. **Variations**:
   - Different distances (close-up to 1m away)
   - Different angles (top-down, 45, side)
   - Different backgrounds (desk, floor, hand-held)
   - Different lighting (natural, indoor, dim)
   - Single box and multiple boxes per frame
   - Different boxes / packaging types

## Directory Structure

```
local_train/medicine_box/
  images/
    train/          # 80% of images
    val/            # 20% of images
  labels/
    train/          # YOLO format .txt files
    val/            # YOLO format .txt files
  data.yaml         # Dataset config
```

## Annotation Format (YOLO)

Each `.txt` file corresponds to an image. Each line is one box:

```
<class_id> <x_center> <y_center> <width> <height>
```

All values normalized to [0, 1]. For `medicine_box`, class_id is always `0`.

## Annotation Tools

- **labelImg** (already at `/Users/dep/Documents/New project/labelImg/`): Set to YOLO format, class list: `medicine_box`
- **Label Studio**: Web-based alternative
- **Roboflow**: Cloud-based with free tier

## Bootstrapping with Synthetic Data

As a fallback, synthetic medicine box images can be generated by compositing box photos onto random backgrounds with augmentations. This can produce a minimum viable dataset without manual annotation.
